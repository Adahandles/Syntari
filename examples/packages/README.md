# Syntari Package Examples

Example package manifests showing different use cases.

## Simple Package (example-simple/)

Minimal package with no dependencies.

## Library Package (example-library/)

A reusable library with dependencies.

## Application Package (example-app/)

Complete application with entry point and build configuration.

## Usage

Each directory contains a `syntari.toml` manifest. To use:

```bash
cd example-simple
syntari pkg install
```
